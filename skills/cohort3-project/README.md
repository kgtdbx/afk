# Cohort 3 Project Skills

Skills installed in the `.claude/skills/` directory of the AI Hero cohort 3 project. Mix of course-provided and custom.

| Skill | Origin | Purpose |
|-------|--------|---------|
| `better-sqlite3-rebuild` | Custom (created during session) | Fixes native module errors by running `pnpm rebuild` |
| `do-work` | Course-provided | End-to-end workflow: plan, implement, validate with typecheck/tests, commit |
| `pnpm-not-found` | Course-provided | Fixes `pnpm command not found` by enabling corepack |
| `prd-to-issues` | Course-provided | Breaks a PRD into vertical-slice GitHub issues with dependency graph |
| `write-a-prd` | Course-provided | Interactive PRD generation from user requirements |
| `write-a-skill` | Course-provided | Generates new Claude Code skills |
| `zod-to-valibot` | Course-provided | Migrates Zod schemas to Valibot |
